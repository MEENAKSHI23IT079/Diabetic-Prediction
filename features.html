Features of the Smart Health System for Diabetes

BMI Calculation

Calculates BMI based on height and weight, and provides health range categorization with recommendations.

Glucose Level Monitoring

Tracks and displays real-time blood glucose levels with alerts, historical data, and personalized recommendations.

Medication Reminder

Sends customizable reminders for medication schedules, tracks intake, and provides refill alerts.

Glucose Data Graph

Visualizes glucose levels over time with interactive charts and overlay options for deeper insights.

User-Friendly Interface

Features an intuitive design, customizable dashboard, and multi-language support for a seamless user experience.

AI Chatbot

Offers instant, personalized support and guidance, answering questions and helping with diabetes management tasks.
